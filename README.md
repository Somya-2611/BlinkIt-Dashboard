# 🛒 BlinkIt-Dashboard

### Project Description

The Blinkit Sales Dashboard is an interactive Power BI report designed to analyze sales performance, customer preferences, and outlet operations across Blinkit's retail network. The dashboard enables users to monitor key business metrics, compare outlet performance, and identify sales trends through dynamic visualizations and filters.

Users can explore data by outlet location, outlet size, and item type to gain valuable insights into revenue generation, product demand, customer ratings, and outlet performance.

### Tech Stack

This project was developed using the following technologies:
- 🐍 **Python** – Data cleaning, preprocessing, exploratory data analysis (EDA), and visualization.
- 🗄️ **SQL (PostgreSQL)** – Business queries, KPI generation, and analytical reporting.
- 📊 **Microsoft Excel** – Data analysis, Pivot Tables, Pivot Charts, and interactive dashboard creation.
- 📈 **Power BI Desktop** – Interactive business intelligence dashboard.
- 📂 **Power Query** – Data transformation and preparation.
- 🧠 **DAX (Data Analysis Expressions)** – Calculated measures and dynamic visualizations.
- 📚 **Pandas, NumPy** – Data manipulation and analysis.
- 📉 **Matplotlib & Seaborn** – Statistical data visualization.
  

# Features / Highlights

## Business Problem

Retail grocery businesses like Blinkit handle thousands of products across multiple outlet locations and sizes. Analyzing sales performance, customer satisfaction, and inventory trends using raw transactional data is time-consuming and makes it difficult to identify high-performing outlets, popular product categories, and areas requiring improvement.

Key business questions such as:

* Which outlet location generates the highest revenue?
* Which outlet size contributes the most sales?
* Which product categories drive the highest sales?
* How do customer ratings vary across outlet types?
* How has outlet performance changed over time?

...are difficult to answer efficiently without an interactive analytical dashboard.

## Goal of the Project

The objective of this project is to build a comprehensive retail analytics solution that:

- Performs data cleaning and preprocessing.
- Extracts business insights using SQL.
- Conducts exploratory data analysis using Python.
- Builds an interactive Excel dashboard.
- Develops a professional Power BI dashboard.
- Supports data-driven business decision making.

## 📈 Power BI Dashboard

### Walkthrough of Key Visuals

### KPI Cards (Top Left)

Displays the most important business metrics at a glance:

* **Total Sales:** $1.20M
* **Average Sales:** $140.99
* **Number of Items:** 8,523
* **Average Rating:** 3.92

These KPIs provide a quick overview of overall business performance.

### Filter Panel

Interactive slicers allow users to filter the dashboard by:
- Outlet Location Type
- Outlet Size
- Item Type
  
All visuals update dynamically based on the selected filters.

### Outlet Establishment Trend (Line Chart)

Shows total sales across outlet establishment years, helping users understand sales trends and identify years with the strongest outlet performance.

### Fat Content Distribution (Donut Chart)

Compares sales generated by:
* Low Fat products
* Regular products

This helps understand customer purchasing preferences based on product fat content.

### Sales by Item Type (Horizontal Bar Chart)

Ranks product categories such as:
* Fruits & Vegetables
* Snack Foods
* Household
* Frozen Foods
* Dairy
  
allowing users to quickly identify the highest-selling product categories.

### Fat Content by Outlet (Stacked Bar Chart)

Compares Low Fat and Regular product sales across different outlet tiers, enabling performance comparison between outlet locations.

### Outlet Size Distribution (Pie Chart)

Displays sales contribution by:
* Small Outlets
* Medium Outlets
* High Outlets
  
This helps evaluate which outlet size contributes the most revenue.

### Outlet Location Performance (Bar Chart)

Compares total sales across:
* Tier 1
* Tier 2
* Tier 3
  
making it easy to identify the best-performing outlet locations.

### Outlet Type Performance (Summary Table)

Provides a detailed comparison of each outlet type based on:
* Total Sales
* Average Sales
* Average Rating
* Number of Items
* Item Visibility
  
This visual offers a comprehensive performance summary for all outlet formats.

## • Business Impact & Insights

* **Sales Performance Monitoring:** Enables managers to monitor key business KPIs in real time.
* **Product Strategy:** Identifies top-selling product categories to optimize inventory and promotional campaigns.
* **Outlet Performance Evaluation:** Helps compare outlet sizes, locations, and outlet types to identify high-performing stores.
* **Customer Insights:** Uses average ratings and purchasing patterns to understand customer preferences.
* **Strategic Decision Making:** Supports data-driven decisions related to expansion, inventory planning, and marketing initiatives through interactive filtering and trend analysis.
